# 自动化任务设计说明

## 任务功能说明

仅演示说明，具体可概据实际情况下更换机器人，随意设计。

1. 机器人  
   查询机器人`58.34.177.162`的一些运行信息，然后执行纯路径导航任务

2. 调度  
   查询`RDS Core`的订单信息（通过指定订单号及分页）。
   通过调度对两辆车`sim_01`,`sim_02`分别下达了运单


## 任务设计器简要说明  

仅简单介绍，详见产品手册  
* 左侧：任务及指令块  
* 中间：任务设计区域   
* 右侧：指令块的属性设置及任务的入参   
* 底栏：任务的问题检测及运行日志及终端输出  

## 如何运行当前设计的任务

这里建议使用调试功能，调试的过程中可一步步看到实际的运行状态

## 如何将设计的任务投入到生产中

`任务设计器`最终生成的是`python`代码。生成的`python`可以跟现有的`python`混用开发。  
生成的`python`代码没有过多的封装性可读性很高。
如果运行，需要本地安装`python3`。

有两种方式:   

1. 选择任务右键，查看`python`代码   

2. 发布到本地，会将当前的工程全部生成可用的`python`代码
